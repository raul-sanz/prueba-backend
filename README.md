
## Description

Api echa en NestJS.

## Installation

```bash
$ yarn install
```

## Implementacion de Mongo
Se deve cambiar el nombre del archivo .env.example a .env y modificar las variables de entorno
```bash
DB_USER=usuario
DB_PASSWORD=contrase√±a
DB_HOST=urldelhost
DB_NAME=basededatos

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
